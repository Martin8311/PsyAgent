# 考前焦虑的识别与应对

考前焦虑是大学生常见的情绪反应，表现为心慌、注意力难以集中、反复担心考不好、入睡困难等。适度的紧张能提升专注力，但过度焦虑会影响发挥，需要主动调节。

## 常见信号

- 一想到考试就心跳加快、手心出汗、胃部不适。
- 反复出现"我肯定考砸"这类灾难化想法。
- 明明复习了却觉得什么都没记住，越复习越慌。
- 考前夜晚难以入睡，或频繁惊醒。

## 可操作的缓解方法

1. 腹式呼吸：吸气四秒、屏息四秒、呼气六秒，重复五轮，帮助身体从应激状态平复下来。
2. 把"我会失败"改写成"我已经准备了一部分，尽力发挥就好"，用现实的语言替换灾难化想法。
3. 把大目标拆成小任务，每完成一项就打勾，用掌控感对抗失控感。
4. 考前一晚不再啃新内容，保证睡眠比多背两道题更重要。
5. 适度运动，如快走或拉伸二十分钟，有助于消耗焦虑带来的过剩能量。

## 什么时候需要求助

如果焦虑持续两周以上、严重影响吃饭睡觉，或出现强烈的自我否定，建议主动联系学校心理咨询中心或信任的老师，寻求专业支持并不丢人。
